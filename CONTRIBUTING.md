# Contributing

Thank you for your interest in the Glass Box Portfolio!

This is a personal portfolio project, so **we kindly do not accept pull requests or contributions**. However, you're more than welcome to:

1. **Fork the repository** and create your own version for your personal portfolio
2. **Use it as inspiration** for building your own explainable AI systems
3. **Open issues** to report bugs or ask questions (we're happy to help!)

## Creating Your Own Version

If you'd like to build your own Glass Box Portfolio:

1. Fork this repository
2. Update the personal data in `backend/app/tools/experience.py`
3. Customize the styling to match your preferences
4. Deploy to your own Vercel/Cloud Run instances

## Development Setup

For those forking the project, see [README.md](README.md) for setup instructions.

## Code Standards

- **Frontend**: ESLint, TypeScript strict mode, Vitest for tests
- **Backend**: Ruff, mypy, pytest
- **Commits**: Use conventional commits (feat:, fix:, docs:)

## Questions?

Feel free to open an issue if you have questions about the architecture or implementation.
